# DNCL

DNCL (Common Test Procedure Description Standard Language) は、日本の大学入学共通テストの「情報の基礎」科目におけるテスト手順を記述するために使用される言語です。

## 機能
- 変数、値、配列をサポート
- 出力文、代入文、各種演算子(算術、比較、論理)を提供
- 条件分岐、ループ構造、関数呼び出しを含む

## 使い方
DNCLは、日本の大学入学共通テストの「情報の基礎」科目におけるテスト手順の記述に使用されます。この言語とその使用法は、付属のドキュメントで定義されています。
