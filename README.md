# DNCL

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

DNCL (Common Test Procedure Description Standard Language) is a language used by the National Center for University Entrance Examinations in Japan for describing test procedures in the "Information Fundamentals" subject.

## Features
- Supports variables, values, and arrays
- Provides output statements, assignment statements, and various arithmetic, comparison, and logical operations
- Includes conditional branching, loop constructs, and function calls

## Usage
DNCL is used to describe test procedures in the "Information Fundamentals" subject of the Common Test for University Admission in Japan. The language and its usage are defined in the accompanying documentation.
